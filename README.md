### ✨ Project Structure

이 레포지토리는 Next.js (App Router)를 기반으로 구축되었으며, 다음과 같은 주요 서비스를 포함하고 있습니다.

1. 📘 Personal Blog

- **Path:** `src/app/(main)`
- 개인적인 생각과 기술적인 배움을 기록하는 공간입니다.

2. 🏛️ Assembly Dining

- **Path:** `src/app/assembly-dining`
- 국회 앞 맛집 정보를 전합니다.
